# D2D1Tan function


## -description


Returns the tangent of an angle.


## -parameters




### -param angle [in]

Type: <b>FLOAT</b>

The angle to calculate the tangent for.


## -returns



Type: <b>FLOAT</b>

The tangent of the angle.
