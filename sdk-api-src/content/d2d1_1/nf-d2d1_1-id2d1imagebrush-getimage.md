# ID2D1ImageBrush::GetImage


## -description


Gets the image associated with the image brush.


## -parameters




### -param image [out]

Type: <b><a href="https://msdn.microsoft.com/9f7b4546-edbe-4000-a4ce-1a69563ebf9d">ID2D1Image</a>**</b>

When this method returns, contains the address of a pointer to the image associated with  this brush.
          


## -returns



This method does not return a value.
