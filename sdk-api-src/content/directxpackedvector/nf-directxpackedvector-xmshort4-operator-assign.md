# XMSHORT4::operator-assign


## -description


Assigns the vector component data from one instance of <code>XMSHORT4</code> to the current
	instance of <code>XMSHORT4</code>.
    

This operator assigns the vector component data from one instance of <a href="https://msdn.microsoft.com/49DAD98F-14D6-45B9-9D30-0EC553BB32EF">XMSHORT4</a> to the current instance of <code>XMSHORT4</code>.
<div class="alert"><b>Note</b>  This operator is only available under C++.
    </div><div> </div>

The current instance of <code>XMSHORT4</code> whose vector component data has been
		updated to match those of the <code>XMSHORT4</code> instance specified by the
		<b>Short4</b> argument.
