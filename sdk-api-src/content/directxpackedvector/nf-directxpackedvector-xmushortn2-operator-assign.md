# XMUSHORTN2::operator-assign


## -description


Assigns the vector component data from one instance of <code>XMUSHORTN2</code> to the current instance of
  <code>XMUSHORTN2</code>.

This operator assigns the vector component data from one instance of <code>XMUSHORTN2</code> to the current instance of
  <code>XMUSHORTN2</code>.
<div class="alert"><b>Note</b>  This operator is only available under C++.</div><div> </div>

## -parameters




### -param arg1

TBD




#### - UShortN2 [ref]

Instance of <code>XMUSHORTN2</code> used to update the current <code>XMUSHORTN2</code> structure.


## -returns



The current instance of <code>XMUSHORTN2</code> whose vector component data has been updated to match 
      those of the <code>XMUSHORTN2</code> instance specified by the UShortN2 argument.
