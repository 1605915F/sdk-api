# XMSHORTN4::operator-assign


## -description


Assigns the vector component data from one instance of <code>XMSHORTN4</code> to the current
	instance of <code>XMSHORTN4</code>.
    

This operator assigns the vector component data from one instance of <a href="https://msdn.microsoft.com/1dfe894e-bc5f-4bfd-a4ba-71fb3e4c9728">XMSHORTN4</a> to the current instance of <code>XMSHORTN4</code>.
<div class="alert"><b>Note</b>  This operator is only available for C++ based development.
    </div><div> </div>

## -parameters




### -param arg1

TBD




#### - UShortN4 [ref]

Instance of <code>XMSHORTN4</code> used to update the current <code>XMSHORTN4</code>structure.
	    


## -returns



The current instance of <code>XMSHORTN4</code> whose vector component data has been
		updated to match those of the <code>XMSHORTN4</code> instance specified by the <b>UShortN4</b> argument.
