# ID2D1CommandSink4 interface


## -description


This interface performs all the same functions as the existing <a href="https://msdn.microsoft.com/DC4C947E-2EC9-44CC-872A-74DAB6E9C2C8">ID2D1CommandSink3</a> interface. 
        It also enables access to the new primitive blend mode, MAX, through the <a href="https://msdn.microsoft.com/20934CEA-2B89-45F5-8E61-CD47C4A9B78F">SetPrimitiveBlend2</a> method.
