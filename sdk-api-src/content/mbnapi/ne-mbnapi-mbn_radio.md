# MBN_RADIO enumeration


## -description


The MBN_RADIO enumerated type indicates whether the device radio is on or off.


## -enum-fields




### -field MBN_RADIO_OFF

The device radio is off.


### -field MBN_RADIO_ON

The device radio is on.
