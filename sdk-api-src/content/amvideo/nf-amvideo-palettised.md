# PALETTISED macro


## -description


The <code>PALETTISED</code> macro checks whether a bitmap has a color depth of 8 bits or less. 


## -parameters




### -param pbmi

Pointer to a <a href="https://msdn.microsoft.com/en-us/library/Dd407325(v=VS.85).aspx">VIDEOINFOHEADER</a> structure.
