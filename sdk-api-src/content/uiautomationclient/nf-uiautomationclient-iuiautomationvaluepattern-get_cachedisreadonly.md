# IUIAutomationValuePattern::get_CachedIsReadOnly


## -description


Retrieves a cached value that indicates whether the value of the element is read-only.

This property is read-only.


## -parameters


## -remarks



This property must be <b>TRUE</b> for <a href="https://msdn.microsoft.com/9b4caa59-bda4-4cc6-b2d8-ff47ea292746">IUIAutomationValuePattern::SetValue</a> to succeed.
