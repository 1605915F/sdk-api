# IDiscFormat2RawCD::get_BufferUnderrunFreeDisabled


## -description


Determines if Buffer Underrun Free recording is enabled.


## -parameters




### -param value [out]

Is VARIANT_TRUE if Buffer Underrun Free recording is disabled; otherwise, VARIANT_FALSE (enabled).
