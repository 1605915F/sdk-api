# DHCP_SCAN_ITEM structure


## -description


The <b>DHCP_SCAN_ITEM</b> structure defines a desynchronized client lease address stored on a DHCPv4 server, and the location in which it should be fixed (in-memory cache or database).


## -struct-fields




### -field IpAddress

DHCP_IP_ADDRESS value that specifies the address whose lease status was changed during a scan operation.


### -field ScanFlag


<a href="https://msdn.microsoft.com/825a0e64-b0c2-453e-8e00-52f84c40bef3">DHCP_SCAN_FLAG</a>enumeration value that indicates whether the supplied client lease IP address will be fixed in the DHCPv4 server's  in-memory client lease cache or the client lease database proper.
