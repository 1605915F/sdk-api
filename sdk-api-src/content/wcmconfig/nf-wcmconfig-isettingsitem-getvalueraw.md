# ISettingsItem::GetValueRaw


## -description


Gets the value from the current item as
    a byte array.
<div class="alert"><b>Note</b>  The caller must release the
    returned byte array by calling CoTaskMemFree.</div><div> </div>

## -parameters




### -param Data [out]

An array of BYTE pointers, allocated with <a href="https://msdn.microsoft.com/en-us/library/ms692727(v=VS.85).aspx">CoTaskMemAlloc</a>, of length DataSize.


### -param DataSize [out]

The length of the data.
