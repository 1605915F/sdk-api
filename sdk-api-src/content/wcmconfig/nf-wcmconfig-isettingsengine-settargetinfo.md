# ISettingsEngine::SetTargetInfo


## -description


Sets the current offline target for the engine.


## -parameters




### -param Target [in]

An <a href="https://msdn.microsoft.com/f1dd3c93-43ca-4804-8330-55acaccf8ea8">ITargetInfo</a> value that specifies the target.


## -returns



This method returns an HRESULT value. <b>S_OK</b> indicates success.
