# PlaneIntersectionType enumeration


## -description


Indicates whether an object intersects a plane.


## -enum-fields




### -field FRONT

The object is in front of the plane.


### -field INTERSECTING

The object intersects the plane.


### -field BACK

The object is behind the plane.
