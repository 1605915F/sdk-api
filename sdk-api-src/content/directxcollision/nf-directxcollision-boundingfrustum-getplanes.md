# BoundingFrustum::GetPlanes


## -description


Gets the planes making up the <a href="https://msdn.microsoft.com/C0C961B6-6F6B-443B-B02F-2601E158F51B">BoundingFrustum</a>.


## -parameters




### -param NearPlane

A vector that will hold the near plane.


### -param FarPlane

A vector that will hold the far plane.


### -param RightPlane

A vector that will hold the right plane.


### -param LeftPlane

A vector that will hold the left plane.


### -param TopPlane

A vector that will hold the top plane.


### -param BottomPlane

A vector that will hold the bottom plane.


## -returns



This method does not return a value.




## -remarks

Microsoft Visual Studio 2010 or Microsoft Visual Studio 2012 with the Windows SDK for Windows 8. Supported for Win32 desktop apps, Windows Store apps, and Windows Phone 8 apps.
