# ITextStrings::SetFormattedText


## -description


Replaces text with formatted text.


## -parameters




### -param pRangeD [in]

Type: <b><a href="https://msdn.microsoft.com/905f0967-8b99-45ed-a1cc-19d49e919a65">ITextRange2</a>*</b>

The text to be replaced.


### -param pRangeS [in]

Type: <b><a href="https://msdn.microsoft.com/905f0967-8b99-45ed-a1cc-19d49e919a65">ITextRange2</a>*</b>

The formatted text.
