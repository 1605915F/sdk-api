# ITextRow::SetCellColorBack


## -description


Sets the background color of the active cell.


## -parameters




### -param Value [in]

Type: <b>long</b>

The background color.


## -returns



Type: <b><a href="https://msdn.microsoft.com/4553cafc-450e-4493-a4d4-cb6e2f274d46">HRESULT</a></b>

If the method succeeds, it returns <b>NOERROR</b>. Otherwise, it returns an <b>HRESULT</b> error code.




## -remarks



 See <a href="https://msdn.microsoft.com/450f97ea-b5b4-44e4-92b8-155c1a9c9c1b">ITextRow::GetCellShading</a> to see how the background color is used together with the foreground color.
