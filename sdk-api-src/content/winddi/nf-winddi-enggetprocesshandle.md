# EngGetProcessHandle function


## -description


The <b>EngGetProcessHandle</b> function retrieves a handle to the current client process.


## -parameters






## -returns



In Windows 2000 and later, the <b>EngGetProcessHandle</b> function always returns <b>NULL</b>.
