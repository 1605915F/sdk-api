# IWMPEvents::SwitchedToPlayerApplication


## -description



The <b>SwitchedToPlayerApplication</b> event occurs when a remoted Windows Media Player control switches to the full mode of the Player.




## -parameters






## -returns



This method does not return a value.




## -remarks



This event occurs only when remoting the Windows Media Player control.

<b>Windows Media Player 10 Mobile: </b>This event is not supported.
