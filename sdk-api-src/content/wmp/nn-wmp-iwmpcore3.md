# IWMPCore3 interface


## -description



The <b>IWMPCore3</b> interface provides methods that supplement the <b>IWMPCore2</b> interface.




## -inheritance

The <b xmlns:loc="http://microsoft.com/wdcml/l10n">IWMPCore3</b> interface inherits from <a href="https://msdn.microsoft.com/5f839bfe-bf67-49eb-8743-57713e1be7c5">IWMPCore2</a>. <b>IWMPCore3</b> also has these types of members:

</ul>
