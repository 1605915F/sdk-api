# IVssEnumObject::Reset


## -description


The <b>Reset</b> method resets the enumerator 
    so that <a href="https://msdn.microsoft.com/9bfaba94-802f-47f5-9843-acc05b32f1b2">IVssEnumObject:Next</a> starts at the first 
    enumerated object.
