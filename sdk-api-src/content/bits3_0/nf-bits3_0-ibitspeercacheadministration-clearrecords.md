# IBitsPeerCacheAdministration::ClearRecords


## -description


Removes all the records and files from the cache.


## -parameters

<th>Return code</th>
<th>Description</th>

</dl>
</td>
<td width="60%">

</td>
</tr>
</table>
 




## -remarks



The cache is not cleared until all current cache activity is complete.
