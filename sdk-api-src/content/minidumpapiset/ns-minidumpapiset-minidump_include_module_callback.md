# MINIDUMP_INCLUDE_MODULE_CALLBACK structure

<a href="https://msdn.microsoft.com/8dc95b0a-6aee-4c38-ab25-a800153bbe91">MiniDumpCallback</a> function when the callback type is 
<b>IncludeModuleCallback</b>.
