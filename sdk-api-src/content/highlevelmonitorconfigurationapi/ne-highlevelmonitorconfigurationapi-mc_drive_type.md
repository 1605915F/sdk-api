# MC_DRIVE_TYPE enumeration


## -description


Specifies whether to set or get a monitor's red, green, or blue drive.


## -enum-fields




### -field MC_RED_DRIVE

Red drive.
          


### -field MC_GREEN_DRIVE

Green drive.
          


### -field MC_BLUE_DRIVE

Blue drive.
