# IMallocSpy interface


## -description


Enables application developers to monitor (spy on) memory allocation, detect memory leaks, and simulate memory failure in calls to <a href="https://msdn.microsoft.com/047f281e-2665-4d6d-9a0b-918cd3339447">IMalloc</a> methods.



## -inheritance

The <b xmlns:loc="http://microsoft.com/wdcml/l10n">IMallocSpy</b> interface inherits from the <a href="https://msdn.microsoft.com/en-us/library/ms680509(v=VS.85).aspx">IUnknown</a> interface. <b>IMallocSpy</b> also has these types of members:
<ul>

</ul>
