# ID3D11ShaderReflection::GetNumInterfaceSlots


## -description


Gets the number of interface slots in a shader.


## -parameters






## -returns



Type: <b><a href="https://msdn.microsoft.com/4553cafc-450e-4493-a4d4-cb6e2f274d46">UINT</a></b>
