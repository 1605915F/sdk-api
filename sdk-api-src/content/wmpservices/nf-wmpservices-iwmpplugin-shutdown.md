# IWMPPlugin::Shutdown


## -description



The <b>IWMPPlugin::Shutdown</b> method is called when Windows Media Player shuts down the plug-in.




## -parameters






## -returns



The method returns an <b>HRESULT</b>.




## -remarks



<b>Init</b> and <b>Shutdown</b> will always be called on the same thread.
