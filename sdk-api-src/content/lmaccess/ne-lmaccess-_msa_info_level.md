# _MSA_INFO_LEVEL enumeration


## -description


The <b>MSA_INFO_LEVEL</b> enumeration indicates the level of a managed service account.


## -enum-fields




### -field MsaInfoLevel0

The default level.


### -field MsaInfoLevelMax

The maximum level.
