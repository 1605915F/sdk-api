# USER_INFO_21 structure


## -description


The
				<b>USER_INFO_21</b> structure contains a one-way encrypted LAN Manager 2.<i>x</i>-compatible password.


## -struct-fields




### -field usri21_password

Specifies a one-way encrypted LAN Manager 2.<i>x</i>-compatible password.
