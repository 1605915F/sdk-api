# REMEDIATIONSERVERPROPERTIES enumeration


## -description


The values of the 
<b>REMEDIATIONSERVERPROPERTIES</b> enumeration type enumerate the properties of a remediation server.


## -enum-fields




### -field PROPERTY_REMEDIATIONSERVER_ADDRESS

IP address value for a Remediation Server. Used in Network Policy Server (NPS) user interface.


### -field PROPERTY_REMEDIATIONSERVER_FRIENDLY_NAME

Host-name for a Remediation Server. Used in NPS user interface.
