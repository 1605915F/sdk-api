# RADIUSSERVERGROUPPROPERTIES enumeration


## -description


The values in the 
<b>RADIUSSERVERGROUPPROPERTIES</b> enumeration type enumerate properties of a RADIUS server group.


## -enum-fields




### -field PROPERTY_RADIUSSERVERGROUP_SERVERS_COLLECTION

The collection of servers in the RADIUS server group.


## -remarks



The 
<a href="https://msdn.microsoft.com/4290621c-7fc7-416b-89a2-11f2254f0d70">IASPROPERTIES</a> enumeration type contains the value PROPERTY_IAS_RADIUSSERVERGROUPS_COLLECTION, which corresponds to the collection of RADIUS server groups.
