# SYSNLS_FUNCTION enumeration


## -description


Specifies NLS function capabilities.


## -enum-fields




### -field COMPARE_STRING

Value indicating comparison of two strings in the manner of the <a href="https://msdn.microsoft.com/4db84fa7-f3c2-48fb-ad7d-8673397c4b0e">CompareString</a> function or <a href="https://msdn.microsoft.com/84dda2cd-cbf9-45e9-b18c-7dea0b5bc991">LCMapString</a> with the LCMAP_SORTKEY flag specified.
