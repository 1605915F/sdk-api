# IS_HIGH_SURROGATE macro


## -description


Determines if a character is a UTF-16 high <a href="https://msdn.microsoft.com/0dea39e2-a2b4-47fc-b44a-56af8ba1e346">surrogate</a> code point, ranging from 0xd800 to 0xdbff, inclusive.


## -parameters




### -param wch

Character to test.
