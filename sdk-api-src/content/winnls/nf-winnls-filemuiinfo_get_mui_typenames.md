# FILEMUIINFO_GET_MUI_TYPENAMES macro


## -description


Gets the MUI module names multistring array associated with the type name offset information in the <b>dwTypeNameMUIOffset</b> member of a <a href="https://msdn.microsoft.com/4c757d19-ac66-4ba4-a691-f575f61961be">FILEMUIINFO</a> structure.


## -parameters




### -param pInfo

Pointer to the <a href="https://msdn.microsoft.com/4c757d19-ac66-4ba4-a691-f575f61961be">FILEMUIINFO</a> structure.
