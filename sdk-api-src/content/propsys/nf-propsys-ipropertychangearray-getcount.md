# IPropertyChangeArray::GetCount


## -description


Gets the number of change operations in the array.


## -parameters




### -param pcOperations [out]

Type: <b>UINT*</b>

A pointer to the number of change operations.


## -returns



Type: <b>HRESULT</b>

Always returns <b>S_OK</b>.
