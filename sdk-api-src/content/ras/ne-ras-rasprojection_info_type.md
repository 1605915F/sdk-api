# RASPROJECTION_INFO_TYPE enumeration


## -description


The <b>RASPROJECTION_INFO_TYPE</b> enumeration  is used in the <a href="https://msdn.microsoft.com/ca4fbaff-f035-4340-8d29-7dcddaf1b1bb">RAS_PROJECTION_INFO</a> structure to represent either a <a href="https://msdn.microsoft.com/8394b843-75f0-4bbd-9ad8-6f4b5dc4bf7b">RASPPP_PROJECTION_INFO</a> or <a href="https://msdn.microsoft.com/c88346f0-118e-4468-83b1-2dfc263e22f7">RASIKEV2_PROJECTION_INFO</a> structure.


## -enum-fields




### -field PROJECTION_INFO_TYPE_PPP

Represents a <a href="https://msdn.microsoft.com/8394b843-75f0-4bbd-9ad8-6f4b5dc4bf7b">RASPPP_PROJECTION_INFO</a> structure.


### -field PROJECTION_INFO_TYPE_IKEv2

Represents a <a href="https://msdn.microsoft.com/c88346f0-118e-4468-83b1-2dfc263e22f7">RASIKEV2_PROJECTION_INFO</a> structure.
