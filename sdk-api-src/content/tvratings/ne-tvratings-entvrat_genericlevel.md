# EnTvRat_GenericLevel enumeration


## -description



This topic applies to Windows XP Service Pack 1 or later.
        



The <b>EnTvRat_GenericLevel</b> enumeration type specifies a generic range of levels that correspond to levels currently in use, or can be used for a custom rating system.
