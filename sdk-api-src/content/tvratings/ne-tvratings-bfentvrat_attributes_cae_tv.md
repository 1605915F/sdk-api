# BfEnTvRat_Attributes_CAE_TV enumeration


## -description



This topic applies to Windows XP Service Pack 1 or later.
        



The <b>BfEnTvRat_Attributes_CAE_TV</b> enumeration type specifies content rating attributes in the English-Canadian television system.


## -enum-fields




### -field CAE_IsBlocked

Block playback.


### -field CAE_ValidAttrSubmask

A bitmask of all valid attribute bits for the English-Canadian language rating system.


## -remarks



This enumeration type maps <a href="https://docs.microsoft.com/previous-versions/windows/desktop/api/tvratings/ne-tvratings-bfentvrat_genericattributes">BfEnTvRat_GenericAttributes</a> flags to the English-Canadian television system. The only valid flag is CAE_IsBlocked.
