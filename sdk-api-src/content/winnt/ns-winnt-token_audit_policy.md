# TOKEN_AUDIT_POLICY structure


## -description


The <b>TOKEN_AUDIT_POLICY</b> structure specifies the per user audit policy for a token.


## -struct-fields




### -field PerUserPolicy

Specifies the per user audit policy for the token.
