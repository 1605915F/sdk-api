# _CLAIM_SECURITY_ATTRIBUTE_FQBN_VALUE structure


## -description


The <b>CLAIM_SECURITY_ATTRIBUTE_FQBN_VALUE</b> structure specifies the fully qualified binary name.


## -struct-fields




### -field Version

The version of the fully qualified binary name value.


### -field Name
