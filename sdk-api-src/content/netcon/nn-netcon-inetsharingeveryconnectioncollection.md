# INetSharingEveryConnectionCollection interface


## -description


<p class="CCE_Message">[Internet Connection Firewall may be altered or unavailable in subsequent versions. Instead, use the <a href="https://msdn.microsoft.com/4043a85f-ebdc-424c-acf5-9097d1472773">Windows Firewall API</a>.]

The 
<b>INetSharingEveryConnectionCollection</b> interface makes it possible for scripting languages such as VBScript and JScript to enumerate all the connections in the connections folder.


## -inheritance

The <b xmlns:loc="http://microsoft.com/wdcml/l10n">INetSharingEveryConnectionCollection</b> interface inherits from the <a href="https://msdn.microsoft.com/en-us/library/ms221608(v=VS.85).aspx">IDispatch</a> interface. <b>INetSharingEveryConnectionCollection</b> also has these types of members:
