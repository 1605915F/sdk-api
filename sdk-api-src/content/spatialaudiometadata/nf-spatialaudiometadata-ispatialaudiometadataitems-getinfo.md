# ISpatialAudioMetadataItems::GetInfo


## -description


Gets the total frame count for the <a href="https://msdn.microsoft.com/54A6B7DE-A41E-4214-AF02-CC19250B9037">ISpatialAudioMetadataItems</a>, which defines valid item offsets.


## -parameters




### -param info [out]

The total frame count, which defines valid item offsets.


## -returns



If the method succeeds, it returns S_OK.
