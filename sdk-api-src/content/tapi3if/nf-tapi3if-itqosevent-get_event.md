# ITQOSEvent::get_Event


## -description


The 
<b>get_Event</b> method gets the 
<a href="https://msdn.microsoft.com/8bf4bfdc-6327-497d-9d19-4771d47982bb">QOS_EVENT</a> indicator.


## -parameters




### -param pQosEvent [out]

Indicator of the QOS event type.
