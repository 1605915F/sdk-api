# ITCallingCard::get_Options


## -description


The 
<b>get_Options</b> method gets the translation options for this address and card.


## -parameters




### -param plOptions [out]

Pointer to 
<a href="https://msdn.microsoft.com/3f5e9952-945e-42b8-8536-b52a0c833282">LINETRANSLATEOPTION</a> flags.
