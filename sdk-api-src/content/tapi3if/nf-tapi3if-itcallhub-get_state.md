# ITCallHub::get_State


## -description


The 
<b>get_State</b> method gets the current state of the CallHub.


## -parameters




### -param pState [out]

Pointer to 
<a href="https://msdn.microsoft.com/8ddfe1f5-2f4a-4b41-83ce-858a669afc31">CALLHUB_STATE</a> indicator of state.
