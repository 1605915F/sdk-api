# ITDigitsGatheredEvent interface


## -description


The 
<b>ITDigitsGatheredEvent</b> interface exposes methods that allow an application to retrieve data when the TAPI Server sends an event indicating that the Server has gathered digits required by the application.


## -inheritance

The <b xmlns:loc="http://microsoft.com/wdcml/l10n">ITDigitsGatheredEvent</b> interface inherits from the <a href="https://msdn.microsoft.com/en-us/library/ms221608(v=VS.85).aspx">IDispatch</a> interface. <b>ITDigitsGatheredEvent</b> also has these types of members:

</ul>
