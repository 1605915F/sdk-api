# WeekDays enumeration


## -description


Defines the days of the week on which to run the data collector set.


## -enum-fields




### -field plaRunOnce

Run only once on the specified start date and time.


### -field plaSunday

Run on Sunday.


### -field plaMonday

Run on Monday.


### -field plaTuesday

Run on Tuesday.


### -field plaWednesday

Run on Wednesday


### -field plaThursday

Run on Thursday.


### -field plaFriday

Run on Friday.


### -field plaSaturday

Run on Saturday.


### -field plaEveryday

Run every day of the week.
