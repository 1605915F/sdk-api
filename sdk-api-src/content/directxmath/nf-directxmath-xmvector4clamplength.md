# XMVector4ClampLength function


## -description


Clamps the length of a 4D vector to a given range.


## -parameters




### -param V [in]

4D vector.


### -param LengthMin [in]

Minimum clamp length.


### -param LengthMax [in]

Maximum clamp length.


## -returns



Returns a 4D vector whose length is clamped to the specified minimum and maximum.




## -remarks
