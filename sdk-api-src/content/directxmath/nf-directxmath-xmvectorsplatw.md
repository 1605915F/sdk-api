# XMVectorSplatW function


## -description


Replicates the w component of a vector to all of the components.


## -parameters




### -param V [in]

Vector from which to select the w component.


## -returns



Returns a vector, all of whose components are equal to the w component of <i>V</i>.




## -remarks

Microsoft Visual Studio 2010 or Microsoft Visual Studio 2012 with the Windows SDK for Windows 8. Supported for Win32 desktop apps, Windows Store apps, and Windows Phone 8 apps.
