# XMVector4EqualInt function


## -description


Tests whether two 4D vectors are equal, treating each component as an unsigned integer.


## -parameters




### -param V1 [in]

4D vector.


### -param V2 [in]

4D vector.


## -returns



Returns true if the 4D vectors are equal and false otherwise.




## -remarks
