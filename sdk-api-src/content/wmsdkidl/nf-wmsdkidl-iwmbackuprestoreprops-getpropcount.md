# IWMBackupRestoreProps::GetPropCount


## -description


<p class="CCE_Message">[<b>GetPropCount</b> is available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in subsequent versions. Instead, use <a href="http://go.microsoft.com/fwlink/p/?linkid=325240">Microsoft PlayReady</a>.
]


The <b>GetPropCount</b> method retrieves the number of properties.



This method is not implemented.
      


## -parameters




### -param pcProps [out]

Pointer to a count of the properties.


## -returns



The method returns E_NOTIMPL.
