# IEnumDot11AdHocInterfaces::Clone


## -description


Creates a new enumeration interface.


## -parameters




### -param ppEnum [out]

A pointer that, on successful return, points to an <a href="https://msdn.microsoft.com/c61bbe3a-ad02-4182-bf10-1ed5fe307bd4">IEnumDot11AdHocInterfaces</a>interface.
