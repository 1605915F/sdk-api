# FWPM_SUBLAYER_ENUM_TEMPLATE0_ structure


## -description


The <b>FWPM_SUBLAYER_ENUM_TEMPLATE0</b> structure is used for enumerating sublayers.


## -struct-fields




### -field providerKey

Uniquely identifies the provider associated with this sublayer. If this value is non-NULL, only options with the specifies provider will be returned.


## -remarks



<b>FWPM_SUBLAYER_ENUM_TEMPLATE0</b> is a specific implementation of FWPM_SUBLAYER_ENUM_TEMPLATE. See <a href="https://msdn.microsoft.com/FBDF53E5-F7DE-4DEB-AC18-6D2BB59FE670">WFP Version-Independent Names and Targeting Specific Versions of Windows</a>  for more information.
