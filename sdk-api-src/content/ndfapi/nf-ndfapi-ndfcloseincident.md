# NdfCloseIncident function


## -description


The <b>NdfCloseIncident</b> function is used to close an Network Diagnostics Framework (NDF) incident following its resolution.


## -parameters




### -param handle [in]

Type: <b>NDFHANDLE</b>

Handle to the NDF incident that is being closed.


## -returns



Type: <b>HRESULT</b>
