# RGNDATAHEADER structure


## -description



The <b>RGNDATAHEADER</b> structure describes the data returned by the <a href="https://msdn.microsoft.com/e0d4862d-a405-4c00-b7b0-af4dd60407c0">GetRegionData</a> function.




## -struct-fields




### -field dwSize

The size, in bytes, of the header.


### -field iType

The type of region. This value must be RDH_RECTANGLES.


### -field nCount

The number of rectangles that make up the region.


### -field nRgnSize

The size of the <a href="https://msdn.microsoft.com/3eac0b23-3138-4b34-9c16-6cc185e4de22">RGNDATA</a> buffer required to receive the <a href="https://msdn.microsoft.com/9439cb6c-f2f7-4c27-b1d7-8ddf16d81fe8">RECT</a> structures that make up the region. If the size is not known, this member can be zero.


### -field rcBound

A bounding rectangle for the region in logical units.
