# GetRValue macro


## -description



The <b>GetRValue</b> macro retrieves an intensity value for the red component of a red, green, blue (RGB) value.




## -parameters




### -param rgb

Specifies an RGB color value.


## -remarks



The intensity value is in the range 0 through 255.
