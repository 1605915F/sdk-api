# IFaxOutgoingQueue::get_Retries


## -description


The <b>IFaxOutgoingQueue::get_Retries</b> property is a value that indicates the number of times that the fax service attempts to retransmit an outgoing fax when the initial transmission fails.

This property is read/write.


## -parameters


## -remarks



To read or to write to this property, a user must have the <a href="https://msdn.microsoft.com/en-us/library/ms689205(v=VS.85).aspx">farQUERY_CONFIG</a> access right.
