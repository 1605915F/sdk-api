# IFaxDocument::get_Priority


## -description


The <b>IFaxDocument::get_Priority</b> property specifies the priority to use when sending the fax; for example, normal, low, or high priority.

This property is read/write.


## -parameters


## -remarks



By default, <b>IFaxDocument::get_Priority</b> is set to <a href="https://msdn.microsoft.com/en-us/library/ms689192(v=VS.85).aspx">fptLow</a>, which indicates low priority.
