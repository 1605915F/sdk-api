# IFaxDocument::get_Priority


## -description


The <b>IFaxDocument::get_Priority</b> property specifies the priority to use when sending the fax; for example, normal, low, or high priority.

This property is read/write.


## -parameters


## -remarks



By default, <b>IFaxDocument::get_Priority</b> is set to <a href="https://msdn.microsoft.com/a7b68ea9-46c8-4509-bb91-8e13851858cb">fptLow</a>, which indicates low priority.
