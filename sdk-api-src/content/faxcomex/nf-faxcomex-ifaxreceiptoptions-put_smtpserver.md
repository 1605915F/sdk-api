# IFaxReceiptOptions::put_SMTPServer


## -description


The <b>IFaxReceiptOptions::get_SMTPServer</b> property is a null-terminated string that contains the name of the Simple Mail Transport Protocol (SMTP) server.

This property is read/write.


## -parameters


## -remarks



To read or to write to this property, a user must have the <a href="https://msdn.microsoft.com/en-us/library/ms689205(v=VS.85).aspx">farQUERY_CONFIG</a> access right.
