# IFaxOutboundRoutingRule::get_UseDevice


## -description


The <b>IFaxOutboundRoutingRule::get_UseDevice</b> property is a Boolean value that indicates whether the outbound routing rule points to a single fax device.

This property is read/write.


## -parameters


## -remarks



To read or to write to this property, a user must have the <a href="https://msdn.microsoft.com/70d729c6-8299-47d7-8dea-f7c754a25531">farQUERY_CONFIG</a> access right.
