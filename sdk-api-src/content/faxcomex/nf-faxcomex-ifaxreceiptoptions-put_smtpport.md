# IFaxReceiptOptions::put_SMTPPort


## -description


The <b>IFaxReceiptOptions::get_SMTPPort</b> property is a value that specifies the Simple Mail Transport Protocol (SMTP) port number.

This property is read/write.


## -parameters


## -remarks



To read or to write to this property, a user must have the <a href="https://msdn.microsoft.com/en-us/library/ms689205(v=VS.85).aspx">farQUERY_CONFIG</a> access right.
