# IFaxDeviceProvider::get_DeviceIds


## -description


The <b>IFaxDeviceProvider::get_DeviceIds</b> property returns a variant safe array of long (VT_I4 | VT_ARRAY). Each long value in the array is a device ID.

This property is read-only.
