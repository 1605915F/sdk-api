# WitnessTagUpdateHelper structure


## -description


Contains information used to update and validate a <a href="https://msdn.microsoft.com/732CB125-F43A-4CC4-BC3F-EFB511BB7F2E">PaxosTagCStruct</a> structure.


## -struct-fields




### -field Version

The cluster configuration version to validate.


### -field paxosToSet

The Paxos tag to update.


### -field paxosToValidate

The Paxos tag to validate.
