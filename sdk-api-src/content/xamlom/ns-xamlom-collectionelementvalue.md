# CollectionElementValue structure


## -description


Represents an element in a collection.


## -struct-fields




### -field Index

The collection index where the element was found.


### -field ValueType

The type of the element object.


### -field Value

The value of the element.


### -field MetadataBits

A bit field representing <a href="https://msdn.microsoft.com/951A4C1F-B176-4D18-821A-CEAD1116B8BE">MetadataBit</a>s.
