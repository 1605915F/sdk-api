# DDBOBNEXTFIELDINFO structure


## -description


The DDBOBNEXTFIELDINFO structure contains the bob information for the surface. 


## -struct-fields




### -field lpSurface

Points to a <a href="https://msdn.microsoft.com/4057cfcf-675e-439f-8b51-23adede1d35a">DDSURFACEDATA</a> structure that represents the surface.
