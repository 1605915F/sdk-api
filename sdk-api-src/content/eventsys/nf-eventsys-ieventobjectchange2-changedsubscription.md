# IEventObjectChange2::ChangedSubscription


## -description


Indicates that a subscription object has been added, modified, or deleted.


## -parameters




### -param pInfo [in]

A pointer to a <a href="https://msdn.microsoft.com/6c9f143e-bdd4-48be-a635-a382c8c770c1">COMEVENTSYSCHANGEINFO</a> structure. 


## -returns



This method can return the standard return values E_INVALIDARG, E_OUTOFMEMORY, E_UNEXPECTED, E_FAIL, and S_OK.
