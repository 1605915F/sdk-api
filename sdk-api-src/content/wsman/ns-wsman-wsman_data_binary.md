# WSMAN_DATA_BINARY structure


## -description


A <a href="https://msdn.microsoft.com/4ff574d4-04b0-47c3-808f-867d6815bffc">WSMAN_DATA</a> structure component that holds binary data for use with various Windows Remote Management functions.


## -struct-fields




### -field dataLength

Represents the number of BYTEs stored in the data field.


### -field data

Specifies the storage location for the binary data.
