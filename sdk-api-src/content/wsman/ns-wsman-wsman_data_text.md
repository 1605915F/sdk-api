# WSMAN_DATA_TEXT structure


## -description


A <a href="https://msdn.microsoft.com/4ff574d4-04b0-47c3-808f-867d6815bffc">WSMAN_DATA</a> structure component that holds textual data for use with various Windows Remote Management functions.


## -struct-fields




### -field bufferLength

Specifies the number of UNICODE characters stored in the buffer.


### -field buffer

Specifies the storage location for the textual data.
