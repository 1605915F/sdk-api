# IInkPresenterDesktop interface


## -description


An <b>IInkPresenterDesktop</b> object represents an <a href="https://msdn.microsoft.com/561e2d14-288a-490a-9a3b-5a32e98b51c3">InkPresenter</a> that can be configured and inserted into the <a href="https://msdn.microsoft.com/40e2d02b-77e8-425f-ac5e-3dcddef08173">DirectComposition</a> visual tree of the Classic Windows app.
