# IDirectDrawSurface7::UpdateOverlayDisplay


## -description


The <b>IDirectDrawSurface7::UpdateOverlayDisplay</b> method is not currently implemented.


## -parameters






#### - dwFlags [in]

The method is not currently implemented.
          


## -returns



The method is not currently implemented.
          




## -remarks



You must use <a href="https://msdn.microsoft.com/d936b4dd-058c-48e1-834b-b47ef6d8ef65">LoadLibrary</a> to explicitly link to Ddraw.dll and then use <a href="https://msdn.microsoft.com/a0d7fc09-f888-4f46-a571-d3719a627597">GetProcAddress</a> to access the <b>UpdateOverlayDisplay</b> method.
