# D3D11_TILE_MAPPING_FLAG enumeration


## -description


Identifies how to perform a tile-mapping operation.


## -enum-fields




### -field D3D11_TILE_MAPPING_NO_OVERWRITE

Indicates that no overwriting of tiles occurs in the tile-mapping operation.
