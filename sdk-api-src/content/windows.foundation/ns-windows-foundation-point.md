# Point structure


## -description


Represents an ordered pair of floating-point x- and y-coordinates that defines a point in a two-dimensional plane.


## -struct-fields




### -field Point.X

 


### -field Point.Y

 




#### - X

Type: <b>float</b>

The horizontal position of the point. 




#### - Y

Type: <b>float</b>

The vertical position of the point.
