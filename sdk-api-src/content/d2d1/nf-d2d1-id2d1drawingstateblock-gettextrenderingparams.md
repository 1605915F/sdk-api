# ID2D1DrawingStateBlock::GetTextRenderingParams


## -description


Retrieves the text-rendering configuration of the drawing state.


## -parameters




### -param textRenderingParams [out]

Type: <b><a href="https://msdn.microsoft.com/28b118e4-9a63-46cf-8ab7-e1039126405b">IDWriteRenderingParams</a>**</b>

When this method returns, contains the address of a pointer to an <a href="https://msdn.microsoft.com/28b118e4-9a63-46cf-8ab7-e1039126405b">IDWriteRenderingParams</a> object that describes the text-rendering configuration of the drawing state.


## -returns



This method does not return a value.
