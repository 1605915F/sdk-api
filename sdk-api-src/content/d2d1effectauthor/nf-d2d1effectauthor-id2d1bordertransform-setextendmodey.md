# ID2D1BorderTransform::SetExtendModeY


## -description


Sets the extend mode in the y direction.


## -parameters




### -param extendMode

Type: <b><a href="https://msdn.microsoft.com/6b6e1fe1-d43a-46cf-904d-5266b9bd6bf4">D2D1_EXTEND_MODE</a></b>

The extend mode in the y direction.


## -returns



This method does not return a value.




## -remarks



If the extend mode enumeration is invalid, this operation is ignored.
