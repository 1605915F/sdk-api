# ID2D1ConcreteTransform::SetCached


## -description


Sets whether the output of the specified transform is cached.


## -parameters




### -param isCached

Type: <b>BOOL</b>

<b>TRUE</b> if the output should be cached; otherwise,  <b>FALSE</b>.


## -returns



This method does not return a value.
