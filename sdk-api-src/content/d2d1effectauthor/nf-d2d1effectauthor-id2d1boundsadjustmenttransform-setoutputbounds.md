# ID2D1BoundsAdjustmentTransform::SetOutputBounds


## -description


This sets the output bounds for the support transform.


## -parameters




### -param outputBounds [in]

Type: <b>const <a href="https://msdn.microsoft.com/655EBC1A-199E-40A2-A4C2-9622AFAEE0B5">D2D1_RECT_L</a>*</b>

The output bounds.


## -returns



This method does not return a value.
