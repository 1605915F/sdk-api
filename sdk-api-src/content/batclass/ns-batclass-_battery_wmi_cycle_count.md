# _BATTERY_WMI_CYCLE_COUNT structure


## -description


Defines information about number of charge cycles of a battery for use with the <a href="https://msdn.microsoft.com/en-us/library/Ff536268(v=VS.85).aspx">BatteryClassQueryWmiDataBlock</a> function.


## -struct-fields




### -field Tag

A tag that identifies a specific battery.


### -field CycleCount

The number of charge/discharge cycles the battery has experienced, or zero if the battery does not support a cycle counter.
