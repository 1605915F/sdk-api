# ID3D10Effect::GetDesc


## -description


Get an effect description.


## -parameters




### -param pDesc [in, out]

Type: <b><a href="https://msdn.microsoft.com/en-us/library/Bb205047(v=VS.85).aspx">D3D10_EFFECT_DESC</a>*</b>

A pointer to an effect description (see <a href="https://msdn.microsoft.com/en-us/library/Bb205047(v=VS.85).aspx">D3D10_EFFECT_DESC</a>).


## -returns



Type: <b><a href="https://msdn.microsoft.com/en-us/library/Hh437604(v=VS.85).aspx">HRESULT</a></b>

Returns one of the following <a href="https://msdn.microsoft.com/en-us/library/Bb205278(v=VS.85).aspx">Direct3D 10 Return Codes</a>.




## -remarks



An effect description contains basic information about an effect such as the techniques it contains and the constant buffer resources it requires.
