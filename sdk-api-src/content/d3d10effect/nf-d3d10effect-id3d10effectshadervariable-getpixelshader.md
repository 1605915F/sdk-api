# ID3D10EffectShaderVariable::GetPixelShader


## -description


Get a pixel shader.


## -parameters




### -param ShaderIndex [in]

Type: <b><a href="https://msdn.microsoft.com/4553cafc-450e-4493-a4d4-cb6e2f274d46">UINT</a></b>

A zero-based index.


### -param ppPS [out]

Type: <b><a href="https://msdn.microsoft.com/en-us/library/Bb173821(v=VS.85).aspx">ID3D10PixelShader</a>**</b>

A pointer to a <a href="https://msdn.microsoft.com/en-us/library/Bb173821(v=VS.85).aspx">ID3D10PixelShader Interface</a>.


## -returns



Type: <b><a href="https://msdn.microsoft.com/en-us/library/Hh437604(v=VS.85).aspx">HRESULT</a></b>

Returns one of the following <a href="https://msdn.microsoft.com/en-us/library/Bb205278(v=VS.85).aspx">Direct3D 10 Return Codes</a>.
