# TfAnchor enumeration


## -description


Elements of the <b>TfAnchor</b> enumeration specify the start anchor or end anchor of an <a href="https://msdn.microsoft.com/b8889f7d-3228-4ecc-8d24-c04234d3101e">ITfRange</a> object.


## -enum-fields




### -field TF_ANCHOR_START

Specifies the start anchor of the <b>ITfRange</b> object.


### -field TF_ANCHOR_END

Specifies the end anchor of the <b>ITfRange</b> object.


## -remarks



A range refers to a span of text in a document. Each range is delimited by a start anchor and an end anchor.
