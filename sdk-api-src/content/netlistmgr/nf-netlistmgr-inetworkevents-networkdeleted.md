# INetworkEvents::NetworkDeleted


## -description


The <b>NetworkDeleted</b> method is called when a network is deleted.


## -parameters




### -param networkId [in]

GUID that contains the network ID of the network that was deleted.


## -returns



Returns S_OK if the method succeeds.
