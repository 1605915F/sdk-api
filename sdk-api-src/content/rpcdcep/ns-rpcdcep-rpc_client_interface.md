# RPC_CLIENT_INTERFACE structure


## -description


The 
<b>RPC_CLIENT_INTERFACE</b> structure is part of the private interface between the run-time libraries and the stubs. Most distributed applications that use Microsoft RPC do not need this structure.

The data structure is defined in the header file Rpcdcep.h. See the header file for syntax block and member definitions.
