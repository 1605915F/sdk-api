# IShellImageData::Scale


## -description


Adjusts the size of an image.


## -parameters




### -param cx [in]

Type: <b>ULONG</b>

The horizontal (x) dimension. If this value is 0, the x dimension is set to a scaled value based on the point specified in <i>cy</i>.


### -param cy [in]

Type: <b>ULONG</b>

The vertical (y) dimension. If this value is 0, the y dimension is set to a scaled value based on the point specified in <i>cx</i>.


### -param hints [in]

Type: <b><a href="https://msdn.microsoft.com/en-us/library/ms534141(v=VS.85).aspx">InterpolationMode</a></b>

A member of the <a href="https://msdn.microsoft.com/en-us/library/ms534141(v=VS.85).aspx">InterpolationMode</a> enumeration, specifying the algorithm that is used when the image is scaled.

Type: <b>HRESULT</b>
