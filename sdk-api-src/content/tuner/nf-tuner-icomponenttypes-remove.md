# IComponentTypes::Remove


## -description



The <b>Remove</b> method removes the <a href="https://msdn.microsoft.com/d5d8af25-4d39-4327-bd6d-8984ae9e6a78">ComponentType</a> object at the specified index number.




## -parameters




### -param Index [in]

Index of the item to remove.


## -returns



Returns S_OK if successful. If the method fails, error information can be retrieved using the standard COM <b>IErrorInfo</b> interface.
