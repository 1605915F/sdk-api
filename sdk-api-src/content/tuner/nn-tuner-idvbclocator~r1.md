# IDVBCLocator interface

## -description

The <b>IDVBCLocator</b> interface is implemented on <a href="https://msdn.microsoft.com/c1b576d4-a16a-4074-8374-bec625500d1c">DVBCLocator</a> objects, which contain tuning information for DVB-C networks.


## -inheritance
IDVBCLocator interits from . 
## -members

	The IDVBCLocator has no additional methods.

## -remarks

To declare the interface identifier (IID) for this interface, use the <b>__uuidof</b> operator: <code>__uuidof(IDVBCLocator)</code>.
