# IFsrmFileManagementJobManager::GetFileManagementJob


## -description


Gets the specified file management job.


## -parameters




### -param name [in]

The name of the file management job to retrieve.

An <a href="https://msdn.microsoft.com/e9ae697d-4f7c-47d9-8d2a-c46c2e5f838f">IFsrmFileManagementJob</a> interface to the specified file management job.
