# eAVEncMPVLevel enumeration


## -description



Specifies the MPEG-2 profile. This enumeration is used with the <a href="https://msdn.microsoft.com/dae29a60-b9f0-47d3-af4c-e08138df3382">AVEncMPVLevel</a> property.




## -enum-fields




### -field eAVEncMPVLevel_Low

Low Level.


### -field eAVEncMPVLevel_Main

Main Level.


### -field eAVEncMPVLevel_High1440

High 1440 Level.


### -field eAVEncMPVLevel_High

High Level.
