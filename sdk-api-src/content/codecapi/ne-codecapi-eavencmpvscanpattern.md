# eAVEncMPVScanPattern enumeration


## -description



Specifies the macroblock scan pattern. This enumeration is used with the <a href="https://msdn.microsoft.com/34d7a2c8-8ac3-4900-9dd9-a7bb7e19b911">AVEncMPVScanPattern</a> property.




## -enum-fields




### -field eAVEncMPVScanPattern_Auto

The encoder selects the scan pattern.


### -field eAVEncMPVScanPattern_ZigZagScan

Zig-zag scan.


### -field eAVEncMPVScanPattern_AlternateScan

Alternate-vertical scan.
