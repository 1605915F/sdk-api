# eAVEncAudioInputContent enumeration


## -description



Specifies whether the audio content contains music or voice. This enumeration is used with the <a href="https://msdn.microsoft.com/10e35d7f-06b8-427f-9309-167358ccbd6f">AVEncAudioInputContent</a> property.




## -enum-fields




### -field AVEncAudioInputContent_Unknown

The audio content is not known.


### -field AVEncAudioInputContent_Voice

The audio contains voice.


### -field AVEncAudioInputContent_Music

The audio contains music.
