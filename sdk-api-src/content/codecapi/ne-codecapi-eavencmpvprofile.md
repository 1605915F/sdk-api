# eAVEncMPVProfile enumeration


## -description



Specifies the MPEG-2 profile. This enumeration is used with the <a href="https://msdn.microsoft.com/89c76f7e-8ec9-414b-8929-7392ff3a11bd">AVEncMPVProfile</a> property.




## -enum-fields




### -field eAVEncMPVProfile_unknown

The profile is not known.


### -field eAVEncMPVProfile_Simple

Simple Profile.


### -field eAVEncMPVProfile_Main

Main Profile.


### -field eAVEncMPVProfile_High

High Profile.


### -field eAVEncMPVProfile_422

4:2:2 Profile.
