# eAVEncVideoFilmContent enumeration


## -description



Specifies whether the original source of the input video was film or video. This enumeration is used with the <a href="https://msdn.microsoft.com/21b0449b-eed0-4879-a938-a7b01b349f22">AVEncVideoSourceFilmContent</a> property.




## -enum-fields




### -field eAVEncVideoFilmContent_VideoOnly

The original source was video.


### -field eAVEncVideoFilmContent_FilmOnly

The original source was film.


### -field eAVEncVideoFilmContent_Mixed

The original source contains a mix of video and film.
