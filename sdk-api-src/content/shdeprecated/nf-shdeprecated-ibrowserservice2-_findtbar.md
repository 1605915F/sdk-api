# IBrowserService2::_FindTBar


## -description


Deprecated. Returns the index of a browser toolbar item based on Component Object Model (COM) identity rules.


## -parameters




### -param punkSrc [in]

Type: <b><a href="https://msdn.microsoft.com/33f1d79a-33fc-4ce5-a372-e08bda378332">IUnknown</a>*</b>

A pointer to the <a href="https://msdn.microsoft.com/33f1d79a-33fc-4ce5-a372-e08bda378332">IUnknown</a> of the browser toolbar item.
        


## -returns



Type: <b>UINT</b>

The index of the browser toolbar item.
