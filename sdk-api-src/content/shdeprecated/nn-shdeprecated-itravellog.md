# ITravelLog interface


## -description


<p class="CCE_Message">[<a href="https://msdn.microsoft.com/b8a5d532-c1fd-4302-b983-cc9a74270321">ITravelEntry</a> may not be supported in versions of Windows later than Windows XP.]
