# RectF::GetLocation


## -description


The <b>RectF::GetLocation</b> method gets the coordinates of the upper-left corner of this rectangle.


## -parameters




### -param point [out]

Type: <b>PointF*</b>

Pointer to a <a href="https://msdn.microsoft.com/en-us/library/ms534488(v=VS.85).aspx">PointF</a> object that receives the coordinates of the upper-left corner of this rectangle. 


## -returns



This method does not return a value.
