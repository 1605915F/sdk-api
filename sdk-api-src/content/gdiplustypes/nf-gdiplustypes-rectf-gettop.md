# RectF::GetTop


## -description


The <b>RectF::GetTop</b> method gets the y-coordinate of the top edge of the rectangle.


## -parameters






## -returns



Type: <strong>Type: <b>REAL</b>
</strong>

This method returns the y-coordinate of the top edge of the rectangle.
