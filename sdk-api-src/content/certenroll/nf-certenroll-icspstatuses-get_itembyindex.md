# ICspStatuses::get_ItemByIndex


## -description


The <b>ItemByIndex</b> property retrieves an <a href="https://msdn.microsoft.com/30cc43c8-6ef3-49ad-8cff-9a5b7389ff68">ICspStatus</a> object from the collection by index number. This property is web enabled.

This property is read-only.
