# ICertPropertyDescription::get_Description


## -description


The <b>Description</b> property retrieves  a description of the certificate. The description identifies the intended use of the certificate.

This property is read-only.


## -parameters


## -remarks



 Call the <a href="https://msdn.microsoft.com/en-us/library/Aa375662(v=VS.85).aspx">Initialize</a> method to create a description.
