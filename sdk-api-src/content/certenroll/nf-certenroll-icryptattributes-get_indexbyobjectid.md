# ICryptAttributes::get_IndexByObjectId


## -description


The <b>IndexByObjectId</b> property retrieves the index of an attribute by <a href="https://msdn.microsoft.com/e6be8932-015e-4058-b249-1671b3fea521">object identifier</a> (OID).

This property is read-only.
