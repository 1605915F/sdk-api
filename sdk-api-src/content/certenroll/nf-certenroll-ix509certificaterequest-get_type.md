# IX509CertificateRequest::get_Type


## -description


The <b>Type</b> property retrieves a  value that specifies the type of the request object.

This property is read-only.


## -parameters


## -remarks



You can use this property with the <a href="https://msdn.microsoft.com/5ade7824-d95a-492d-aadf-487422386500">GetInnerRequest</a> method to determine the type of the inner request object.
