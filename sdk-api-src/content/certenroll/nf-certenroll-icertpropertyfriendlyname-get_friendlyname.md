# ICertPropertyFriendlyName::get_FriendlyName


## -description


The <b>FriendlyName</b> property retrieves the display name of the certificate.

This property is read-only.


## -parameters


## -remarks



 Call the <a href="https://msdn.microsoft.com/en-us/library/Aa375721(v=VS.85).aspx">Initialize</a> method to create a description.
