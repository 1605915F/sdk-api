# IX509Extensions::get_IndexByObjectId


## -description


The <b>IndexByObjectId</b> property retrieves the index of an extension in the collection by <a href="https://msdn.microsoft.com/en-us/library/ms721599(v=VS.85).aspx">object identifier</a> (OID).

This property is read-only.
