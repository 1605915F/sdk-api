# ICertPropertyArchivedKeyHash::get_ArchivedKeyHash


## -description


The <b>ArchivedKeyHash</b> property retrieves a SHA-1 hash of the private key.

This property is read-only.


## -parameters


## -remarks



Call the <a href="https://msdn.microsoft.com/en-us/library/Aa375243(v=VS.85).aspx">Initialize</a> method to specify the hash.
