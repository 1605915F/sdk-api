# ICertPropertyArchivedKeyHash::get_ArchivedKeyHash


## -description


The <b>ArchivedKeyHash</b> property retrieves a SHA-1 hash of the private key.

This property is read-only.


## -parameters


## -remarks



Call the <a href="https://msdn.microsoft.com/1f201b37-6f3a-4f1c-83b8-2f1dbb1d4d07">Initialize</a> method to specify the hash.
