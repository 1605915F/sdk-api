# _OPM_DPCP_PROTECTION_LEVEL enumeration


## -description


Specifies the protection level for DisplayPort Content Protection (DPCP).


## -enum-fields




### -field OPM_DPCP_OFF

DPCP is disabled.


### -field OPM_DPCP_ON

DPCP is enabled.


### -field OPM_DPCP_FORCE_ULONG

Reserved.
