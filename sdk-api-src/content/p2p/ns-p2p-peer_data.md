# PEER_DATA structure


## -description


The <b>PEER_DATA</b> structure contains binary data.


## -struct-fields




### -field cbData

Size of  <b>pbData</b>, in bytes. It is possible for this value to be set to zero if <b>pbData</b> contains no data.


### -field pbData

Pointer to a buffer.
