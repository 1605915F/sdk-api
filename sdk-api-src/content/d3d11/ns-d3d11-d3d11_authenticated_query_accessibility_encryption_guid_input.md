# D3D11_AUTHENTICATED_QUERY_ACCESSIBILITY_ENCRYPTION_GUID_INPUT structure


## -description


Contains input data for a <b>D3D11_AUTHENTICATED_QUERY_ENCRYPTION_WHEN_ACCESSIBLE_GUID</b> query.


## -struct-fields




### -field Input

A <a href="https://msdn.microsoft.com/D1FE4B31-A29D-4079-ABAE-8EB7DB0A0B42">D3D11_AUTHENTICATED_QUERY_INPUT</a> structure that contains the GUID for the query and other data.


### -field EncryptionGuidIndex

The index of the encryption GUID.
