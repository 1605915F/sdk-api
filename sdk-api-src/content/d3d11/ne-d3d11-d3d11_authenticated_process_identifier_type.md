# D3D11_AUTHENTICATED_PROCESS_IDENTIFIER_TYPE enumeration


## -description


Specifies the type of process that is identified in the <a href="https://msdn.microsoft.com/0668B546-6825-4AD9-85CF-CA238028B2E3">D3D11_AUTHENTICATED_QUERY_RESTRICTED_SHARED_RESOURCE_PROCESS_OUTPUT</a> structure.


## -enum-fields




### -field D3D11_PROCESSIDTYPE_UNKNOWN

Unknown process type.


### -field D3D11_PROCESSIDTYPE_DWM

Desktop Window Manager (DWM) process.


### -field D3D11_PROCESSIDTYPE_HANDLE

Handle to a process.
