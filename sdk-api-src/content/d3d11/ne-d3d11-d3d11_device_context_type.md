# D3D11_DEVICE_CONTEXT_TYPE enumeration


## -description


Device context options.


## -enum-fields




### -field D3D11_DEVICE_CONTEXT_IMMEDIATE

The device context is an immediate context.


### -field D3D11_DEVICE_CONTEXT_DEFERRED

The device context is a deferred context.


## -remarks



This enumeration is used by <a href="https://msdn.microsoft.com/fefe2cd7-26c1-4165-9c94-8843571f8824">ID3D11DeviceContext::GetType</a>.
