# D3D12_DEPTH_STENCIL_VALUE structure


## -description


Specifies a depth and stencil value.


## -struct-fields




### -field Depth

Specifies the depth value.


### -field Stencil

Specifies the stencil value.


## -remarks



This structure is used in the <a href="https://msdn.microsoft.com/03B67F91-C150-4719-8C43-D04F51DC9C06">D3D12_CLEAR_VALUE</a> structure.
