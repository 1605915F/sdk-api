# THUMBBUTTONMASK enumeration


## -description


Used by the <a href="https://msdn.microsoft.com/c13657b2-5b96-45ae-b339-b06b13aca65d">THUMBBUTTON</a> structure to specify which members of that structure contain valid data.


## -enum-fields




### -field THB_BITMAP

The <b>iBitmap</b> member contains valid information.


### -field THB_ICON

The <b>hIcon</b> member contains valid information.


### -field THB_TOOLTIP

The <b>szTip</b> member contains valid information.


### -field THB_FLAGS

The <b>dwFlags</b> member contains valid information.
