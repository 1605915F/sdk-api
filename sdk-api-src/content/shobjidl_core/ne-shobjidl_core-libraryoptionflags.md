# LIBRARYOPTIONFLAGS enumeration


## -description


Specifies the library options.


## -enum-fields




### -field LOF_DEFAULT

No library options are set.


### -field LOF_PINNEDTONAVPANE

Pin the library to the navigation pane.


### -field LOF_MASK_ALL

All valid library options flags.
