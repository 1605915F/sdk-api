# IAMWstDecoder::GetDecoderLevel


## -description



Applications use the <code>GetDecoderLevel</code> method to retrieve the WST decoder level. This method is not implemented and always returns AM_WST_LEVEL_1_5.




## -parameters




### -param lpLevel [out]

Receives a member of the <a href="https://msdn.microsoft.com/en-us/library/Dd373507(v=VS.85).aspx">AM_WST_LEVEL</a> enumeration, indicting the decoder level.


## -returns



When the method succeeds, it returns S_OK. Otherwise, it returns an <b>HRESULT</b> error code.
