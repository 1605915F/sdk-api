# IFileSystemImageResult::get_ImageStream


## -description


Retrieves the burn image stream.


## -parameters




### -param pVal [out]

An <a href="https://msdn.microsoft.com/c6f60e37-eadc-46a1-94f6-cacc23613531">IStream</a> interface of the burn image.
