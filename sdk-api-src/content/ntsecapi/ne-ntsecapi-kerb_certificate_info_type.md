# KERB_CERTIFICATE_INFO_TYPE enumeration


## -description


The <b>KERB_CERTIFICATE_INFO_TYPE</b> enumeration specifies the type of certificate information that is provided.


## -enum-fields




### -field CertHashInfo

Identifies certificate hash information as defined by the <a href="https://msdn.microsoft.com/09D78E91-5873-481D-A5FC-B7F39F8F9BB8">KERB_CERTIFICATE_HASHINFO</a> structure.
