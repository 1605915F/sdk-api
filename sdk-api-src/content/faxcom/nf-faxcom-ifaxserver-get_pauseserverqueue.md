# IFaxServer::get_PauseServerQueue


## -description


Sets or retrieves the <b>PauseServerQueue</b> property for a <a href="https://msdn.microsoft.com/en-us/library/ms692367(v=VS.85).aspx">FaxServer</a> object. The <b>PauseServerQueue</b> property is a Boolean value that indicates whether the fax server has paused the fax job queue.

This property is read/write.


## -parameters


## -remarks



An administrator might pause the fax job queue for various reasons. For example, the administrator may need to perform queue management, or free a fax device for an inbound reception or for another application's use.
