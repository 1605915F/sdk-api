# IMtsGrp::get_Count


## -description


Retrieves the number of running packages in the catalog.


## -parameters




### -param pVal [out]

The number of running packages.


## -returns



This method can return the standard return values E_INVALIDARG, E_OUTOFMEMORY, E_FAIL, and S_OK.
