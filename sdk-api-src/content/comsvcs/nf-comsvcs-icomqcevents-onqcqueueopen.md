# IComQCEvents::OnQCQueueOpen


## -description


Generated when a queued components queue is opened. This method is used to generate the <i>QueueID</i> parameter.



## -parameters




### -param pInfo [in]

A pointer to a <a href="https://msdn.microsoft.com/en-us/library/ms688276(v=VS.85).aspx">COMSVCSEVENTINFO</a> structure.


### -param szQueue [in]

The name of the queue.


### -param QueueID [in]

The unique identifier for the queue.


### -param hr [in]

The status from Message Queuing queue open.


## -returns



The user verifies the return values from this method.
