# IInkEdit::get_Hwnd


## -description


Gets a handle to the <a href="https://msdn.microsoft.com/52761cb2-4433-4824-ba19-fe597de2faf0">InkEdit</a> control.

This property is read-only.


## -parameters


## -remarks



<div class="alert"><b>Note</b>  Because the value of this property can change while a program is running, never store the Hwnd value in a variable.
