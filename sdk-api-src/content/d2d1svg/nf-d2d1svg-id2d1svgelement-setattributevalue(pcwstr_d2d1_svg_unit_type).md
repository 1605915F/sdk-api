# ID2D1SvgElement::SetAttributeValue(PCWSTR,D2D1_SVG_UNIT_TYPE)


## -description


Sets an attribute of this element as a unit type value. This method can be used to set the value of a gradientUnits or clipPathUnits attribute.


## -parameters




### -param name [in]

Type: <b>PCWSTR</b>

Name of the attribute to set.


### -param value

Type: <b><a href="https://msdn.microsoft.com/EF7CD98A-AE8D-42B3-BA1F-D50EB2FB88A6">D2D1_SVG_UNIT_TYPE</a></b>

The new value of the attribute.


## -returns



Type: <b><a href="https://msdn.microsoft.com/en-us/library/Hh437604(v=VS.85).aspx">HRESULT</a></b>

This method returns an HRESULT success or error code.
