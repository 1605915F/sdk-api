# ID2D1SvgElement::SetAttributeValue(PCWSTR,D2D1_SVG_LINE_JOIN)


## -description


Sets an attribute of this element as a line join value. This method can be used to set the value of the stroke-linejoin property.


## -parameters




### -param name [in]

Type: <b>PCWSTR</b>

Name of the attribute to set.


### -param value

Type: <b><a href="https://msdn.microsoft.com/C0D506E4-529D-41B1-AB1E-64E12526C7CD">D2D1_SVG_LINE_JOIN</a></b>

The new value of the attribute.


## -returns



Type: <b><a href="455d07e9-52c3-4efb-a9dc-2955cbfd38cc">HRESULT</a></b>

This method returns an HRESULT success or error code.
