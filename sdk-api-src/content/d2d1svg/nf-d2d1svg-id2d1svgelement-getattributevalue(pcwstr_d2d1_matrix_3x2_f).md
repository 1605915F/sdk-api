# ID2D1SvgElement::GetAttributeValue(PCWSTR,D2D1_MATRIX_3X2_F)


## -description


Gets an attribute of this element as a matrix value. This method can be used to get the value of a transform or gradientTransform attribute.


## -parameters




### -param name [in]

Type: <b>PCWSTR</b>

The name of the attribute.


### -param value [out]

Type: <b>D2D1_MATRIX_3X2_F*</b>

The value of the attribute.


## -returns



Type: <b><a href="https://msdn.microsoft.com/en-us/library/Hh437604(v=VS.85).aspx">HRESULT</a></b>

This method returns an HRESULT success or error code.
