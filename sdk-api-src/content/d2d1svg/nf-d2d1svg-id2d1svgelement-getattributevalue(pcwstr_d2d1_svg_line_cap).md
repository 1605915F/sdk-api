# ID2D1SvgElement::GetAttributeValue(PCWSTR,D2D1_SVG_LINE_CAP)


## -description


Gets an attribute of this element as a line cap value. This method can be used to get the value of the stroke-linecap property.


## -parameters




### -param name [in]

Type: <b>PCWSTR</b>

The name of the attribute.


### -param value [out]

Type: <b><a href="https://msdn.microsoft.com/9A44EF41-E2D6-4647-B5EB-D99F14ADFA2F">D2D1_SVG_LINE_CAP</a>*</b>

The value of the attribute.


## -returns



Type: <b><a href="455d07e9-52c3-4efb-a9dc-2955cbfd38cc">HRESULT</a></b>

This method returns an HRESULT success or error code.
