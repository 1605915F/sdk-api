# WS_UINT64_DESCRIPTION structure


## -description


An optional type description used with <a href="https://msdn.microsoft.com/eb3732fd-1197-4e1c-b5b5-9a34aaa0951e">WS_UINT64_TYPE</a> to specify constraints on the set of values
                which can be deserialized.
            


## -struct-fields




### -field minValue

The minimum value.
                


### -field maxValue

The maximum value.
