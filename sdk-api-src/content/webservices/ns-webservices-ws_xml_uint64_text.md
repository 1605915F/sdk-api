# WS_XML_UINT64_TEXT structure


## -description


Represents an unsigned 64 bit integer.  (e.g. The value 255 represents the text "255")
      


## -struct-fields




### -field text

The base type for all types that derive from <a href="https://msdn.microsoft.com/430edd13-b664-4e10-8d61-ffa6a01dcb90">WS_XML_TEXT</a>.
        


### -field value

The unsigned integer value.
