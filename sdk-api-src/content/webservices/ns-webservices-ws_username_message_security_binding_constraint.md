# WS_USERNAME_MESSAGE_SECURITY_BINDING_CONSTRAINT structure


## -description


A security binding constraint that corresponds to the
                <a href="https://msdn.microsoft.com/en-us/library/Dd323497(v=VS.85).aspx">WS_USERNAME_MESSAGE_SECURITY_BINDING</a>.
            


## -struct-fields




### -field bindingConstraint

The base binding constraint that this binding constraint derives from.
                

There are no binding-specific properties are defined for this binding constraint
                    at this time.
                


### -field bindingUsage

This specifies how the security token corresponding to this security binding 
                    should be attached to a message.
