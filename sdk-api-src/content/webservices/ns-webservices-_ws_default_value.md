# _WS_DEFAULT_VALUE structure


## -description


Defines a default value for a field.  This is used
                in a <a href="https://msdn.microsoft.com/8b562fab-f3c5-4732-b993-f7f61ca14ab6">WS_FIELD_DESCRIPTION</a>.
            


## -struct-fields




### -field value

A pointer to the default value.
                


### -field valueSize

The size of the default value, in bytes.
