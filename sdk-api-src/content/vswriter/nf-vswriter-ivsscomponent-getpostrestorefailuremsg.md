# IVssComponent::GetPostRestoreFailureMsg


## -description


The 
<b>GetPostRestoreFailureMsg</b> method returns the failure message generated by a writer while handling the 
<a href="https://msdn.microsoft.com/01cf3931-59ef-4572-9f2e-aa210da0ac2d">PostRestore</a> event, if 
<a href="https://msdn.microsoft.com/1059a586-69e2-4a02-8f52-b8da3f04f51c">IVssComponent::SetPostRestoreFailureMsg</a> set one.

Either a writer or a requester can call this method.


## -parameters




### -param pbstrPostRestoreFailureMsg [out]

Pointer to a string containing the failure message that describes an error that occurred while processing the 
<a href="https://msdn.microsoft.com/01cf3931-59ef-4572-9f2e-aa210da0ac2d">PostRestore</a> event.

## -remarks



The caller should free the memory held by the <i>pbstrPostRestoreFailureMsg</i> parameter by calling <a href="https://msdn.microsoft.com/en-us/library/ms221481(v=VS.85).aspx">SysFreeString</a>.

If 
<a href="https://msdn.microsoft.com/1059a586-69e2-4a02-8f52-b8da3f04f51c">SetPostRestoreFailureMsg</a> was not used to set a 
<a href="https://msdn.microsoft.com/01cf3931-59ef-4572-9f2e-aa210da0ac2d">PostRestore</a> failure message, 
<a href="https://msdn.microsoft.com/58c007a8-8414-4e2d-8042-d249a723780a">GetPreRestoreFailureMsg</a> returns S_FALSE.
