# ADSPEC structure


## -description


The 
<b>ADSPEC</b> structure contains Adspec message information for RSVP.


## -struct-fields




### -field adspec_header

Adspec header, expressed as an <a href="https://msdn.microsoft.com/en-us/library/Aa374125(v=VS.85).aspx">RsvpObjHdr</a> structure.


### -field adspec_body

Adspec message body.
