# IntServMainHdr structure


## -description


The 
<b>IntServMainHdr</b> structure is a header for Integrated Services RSVP objects.


## -struct-fields




### -field ismh_version

Version of this header.


### -field ismh_unused

Reserved. Do not use.


### -field ismh_len32b

Number of 32-bit WORDs in the object, excluding this header object.
