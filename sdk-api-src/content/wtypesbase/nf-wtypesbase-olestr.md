# OLESTR macro


## -description


Transforms string literals into Unicode strings.


## -parameters




### -param str

A string literal.


## -remarks



This macro is defined as follows:

<pre class="syntax" xml:space="preserve"><code>#define OLESTR(str) L##str
</code></pre>
