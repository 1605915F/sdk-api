# Color::GetBlue


## -description


The <b>Color::GetBlue</b> method gets the blue component of this <a href="https://msdn.microsoft.com/en-us/library/ms534427(v=VS.85).aspx">Color</a> object.


## -parameters






## -returns



Type: <strong>Type: <b>BYTE</b>
</strong>

This method returns the blue component of this color.
