# IMSVidRect::put_Width


## -description


The <b>put_Width</b> method specifies the width of the rectangle.


## -parameters




### -param WidthVal [in]

Specifies the width, in pixels.


## -returns



If the method succeeds, it returns S_OK. If it fails, it returns an error code.
