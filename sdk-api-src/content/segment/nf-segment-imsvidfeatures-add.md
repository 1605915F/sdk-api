# IMSVidFeatures::Add


## -description


The <b>Add</b> method adds a feature to the collection.


## -parameters




### -param pDB [in]

Specifies a pointer to the feature's <a href="https://msdn.microsoft.com/0512e1d6-e10e-421e-846c-4bcd7e86d0e7">IMSVidFeature</a> interface.
