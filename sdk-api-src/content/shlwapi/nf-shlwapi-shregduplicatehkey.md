# SHRegDuplicateHKey function


## -description


Duplicates a registry key's HKEY handle.


## -parameters




### -param hkey [in]

Type: <b>HKEY</b>

The HKEY handle to be duplicated.


## -returns



Type: <b>HKEY</b>

Returns a duplicate of the handle specified in <i>hkey</i>.
