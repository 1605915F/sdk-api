# IAMLine21Decoder::SetOutputFormat


## -description



The <code>SetOutputFormat</code> method sets the <a href="https://msdn.microsoft.com/48fa5484-1f8c-4133-b2e1-888cb1834402">Line 21 Decoder</a> filter's output format.



This method is currently not implemented.


## -parameters




### -param lpbmi

Pointer to a <b>BITMAPINFO</b> structure that describes the output format.


## -returns



Returns E_NOTIMPL.
