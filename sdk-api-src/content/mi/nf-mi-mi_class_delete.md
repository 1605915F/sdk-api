# MI_Class_Delete function


## -description


Deletes an <a href="https://msdn.microsoft.com/7f02e0fa-9e58-455d-9cf4-1d1244c44422">MI_Class</a> object.


## -parameters




### -param self [in, out]

A pointer to the class to be deleted.


## -returns



This function returns MI_INLINE MI_Result.
