# MI_ParameterSet_GetParameterCount function


## -description


Gets the number of parameters in a method's parameter set.


## -parameters




### -param self [in]


<a href="https://msdn.microsoft.com/14b5773c-4741-453b-824a-226aab9b8a10">MI_ParameterSet</a> structure.


### -param count [out]

Returned number of parameters.


## -returns



This function returns MI_INLINE MI_Result.
