# _POINTL structure


## -description


The POINTL structure defines the x- and y-coordinates of a point.


## -struct-fields




### -field x

Specifies the <i>x</i>-coordinate of the point. 


### -field y

Specifies the <i>y</i>-coordinate of the point. 


## -remarks



The POINTL structure is identical to the <a href="https://msdn.microsoft.com/6fb3bd3f-f810-4043-812b-94495f38a6aa">POINT</a> structure.
