# IUpdateInstaller4::Commit


## -description


Finalizes updates that were previously staged or installed.


## -parameters




### -param dwFlags

Reserved for future use.


## -returns



Returns <b>S_OK</b> if successful. Otherwise, returns a COM or Windows 

error code.
