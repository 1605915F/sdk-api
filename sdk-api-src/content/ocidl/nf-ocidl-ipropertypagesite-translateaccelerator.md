# IPropertyPageSite::TranslateAccelerator


## -description


Passes a keystroke to the property frame for processing.


## -parameters




### -param pMsg [in]

A pointer to the <a href="https://msdn.microsoft.com/en-us/library/ms644958(v=VS.85).aspx">MSG</a> structure to be processed.
