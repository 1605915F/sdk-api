# SERVER_INFO_1509 structure


## -description


The
				<b>SERVER_INFO_1509</b> structure specifies the maximum raw mode buffer size.


## -struct-fields




### -field sv1509_maxrawbuflen

Specifies the maximum raw mode buffer size, in bytes.
