# IAMTVAudio::UnRegisterNotificationCallBack


## -description



The <code>UnRegisterNotificationCallBack</code> method unregisters an object for event notifications.



This method is not implemented.


## -parameters




### -param pNotify [in]

Pointer to the <a href="https://msdn.microsoft.com/8e5fde62-d17c-4d3c-bdb0-0748a9bd285b">IAMTunerNotification</a> interface that was specified in a previous call to <b>RegisterNotificationCallBack</b>.


## -returns



Returns E_NOTIMPL.
