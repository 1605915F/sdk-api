# IDvdCmd::WaitForEnd


## -description



The <code>WaitForEnd</code> method blocks the DVD Navigator until the command associated with this object completes or is canceled.




## -parameters






## -returns



Returns S_OK if successful or an <b>HRESULT</b> error code otherwise.
