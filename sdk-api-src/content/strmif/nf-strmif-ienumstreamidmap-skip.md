# IEnumStreamIdMap::Skip


## -description



The <code>Skip</code> method skip the element at the specified index.




## -parameters




### -param cRecords [in]

Index of the element to skip.


## -returns



Returns S_OK if successful. If the method fails, an <b>HRESULT</b> error code is returned.
