# _AM_DVD_STREAM_FLAGS enumeration


## -description



Describes a DVD stream type (video, audio, or subpicture).




## -enum-fields




### -field AM_DVD_STREAM_VIDEO

DVD video stream.
          


### -field AM_DVD_STREAM_AUDIO

DVD audio stream.
          


### -field AM_DVD_STREAM_SUBPIC

DVD subpicture stream.
