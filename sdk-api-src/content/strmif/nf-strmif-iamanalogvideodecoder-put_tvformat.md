# IAMAnalogVideoDecoder::put_TVFormat


## -description



The <code>put_TVFormat</code> method sets the analog video format.




## -parameters




### -param lAnalogVideoStandard [in]

Specifies the video format as a member of the <a href="https://msdn.microsoft.com/6760a40c-550c-4774-a5d1-d7e2a6aa6096">AnalogVideoStandard</a> enumeration.


## -returns



Returns S_OK if successful, or an error code otherwise.
