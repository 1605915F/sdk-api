# IWSDUdpAddress::SetSockaddr


## -description


Sets the socket address information.


## -parameters




### -param pSockAddr [in]

Pointer to a <a href="https://msdn.microsoft.com/dfd84b91-0a94-4fe6-b8d2-18562afb9c24">SOCKADDR_STORAGE</a> structure.
