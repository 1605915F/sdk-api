# Matrix4x4F::Scale


## -description


Scales the perspective plane of the matrix.


## -parameters




### -param x

Type: <b>FLOAT</b>

The scale in the X direction.


### -param y

Type: <b>FLOAT</b>

The scale in the Y direction.


### -param z

Type: <b>FLOAT</b>

The scale in the Z direction.


## -returns



Type: <b><a href="https://msdn.microsoft.com/en-us/library/Hh848004(v=VS.85).aspx">Matrix4x4F</a></b>

The result matrix.
