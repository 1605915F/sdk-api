# Matrix4x4F::RotationX


## -description


Rotates the transform matrix around the X axis.


## -parameters




### -param degreeX

Type: <b>FLOAT</b>

The amount of rotation.


## -returns



Type: <b><a href="https://msdn.microsoft.com/113861DF-2E6D-4930-82DC-AA592882E21C">Matrix4x4F</a></b>
