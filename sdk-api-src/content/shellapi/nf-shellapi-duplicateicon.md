# DuplicateIcon function


## -description


Creates a duplicate of a specified icon.


## -parameters




### -param hInst [in]

Type: <b>HINSTANCE</b>


### -param hIcon [in]

Type: <b>HICON</b>

Handle to the icon to be duplicated.


## -returns



Type: <b>HICON</b>

If successful, the function returns the handle to the new icon that was created; otherwise, <b>NULL</b>.




## -remarks



When it is no longer needed, the caller is responsible for freeing the icon handle returned by <b>DuplicateIcon</b> by calling the <a href="https://msdn.microsoft.com/en-us/library/ms648063(v=VS.85).aspx">DestroyIcon</a> function.
