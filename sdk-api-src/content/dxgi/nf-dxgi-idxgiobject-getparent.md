# IDXGIObject::GetParent


## -description


Gets the parent of the object.


## -parameters




### -param riid [in]

Type: <b>REFIID</b>

The ID of the requested interface.


### -param ppParent [out]

Type: <b>void**</b>

The address of a pointer to the parent object.


## -returns



Type: <b><a href="https://msdn.microsoft.com/en-us/library/Hh437604(v=VS.85).aspx">HRESULT</a></b>

Returns one of the <a href="https://msdn.microsoft.com/en-us/library/Bb509553(v=VS.85).aspx">DXGI_ERROR</a> values.
