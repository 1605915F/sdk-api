# CROSS_SLIDE_PARAMETER structure


## -description


Defines the cross-slide threshold and its distance threshold.


## -struct-fields




### -field threshold

One of the constants from <a href="https://msdn.microsoft.com/c498e2ae-a761-4f93-9ec2-4030b03e64b9">CROSS_SLIDE_THRESHOLD</a>.


### -field distance

The <i>threshold</i> distance, in DIPs.
