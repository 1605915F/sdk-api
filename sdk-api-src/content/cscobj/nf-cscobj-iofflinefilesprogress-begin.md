# IOfflineFilesProgress::Begin


## -description


Reports that an operation has begun.


## -parameters




### -param pbAbort [out]

Set this value to <b>TRUE</b> to cancel the operation.   Set to <b>FALSE</b> to continue.


## -returns



The return value is ignored.
