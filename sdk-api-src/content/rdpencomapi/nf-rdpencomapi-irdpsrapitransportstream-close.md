# IRDPSRAPITransportStream::Close


## -description


Called by the Remote Desktop Protocol (RDP) stack to close the stream.


## -parameters






## -returns



If the method succeeds, the return value is <b>S_OK</b>. Otherwise, the return value is an error code.
